# Do-while-loop-
public class DoWhile {
    public static void main(String[] args) {
        int i = 1;

        // This loop will run at least once
        do {
            System.out.println("Iteration " + i);
            i++;
        } while (i <= 5);
    }
}

Output 

Iteration 1
Iteration 2
Iteration 3
Iteration 4
Iteration 5
